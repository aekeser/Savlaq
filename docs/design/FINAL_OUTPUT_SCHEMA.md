# Savlaq Final Output Schema

## Unit of observation

One row = one measured interval from one recording.

## Required identifier columns

speaker_id
speaker_sex
source
language
stimulus_id
recording_file
tier
task_region
interval_label

## Timing columns

start_time
end_time
duration

## Acoustic columns

mean_f0
min_f0
max_f0
mean_intensity
f1
f2

## Metadata columns

word
carrier_sentence
lexical_category
frequency
frequency_rank
syllable_count
vowel_pattern

## QC columns

qc_flag
qc_notes
