# Savlaq Filename Convention

## Pattern

speaker_id_language_source_stimulus_id.wav

## Example

M001_EN_HUMAN_EN_ADJ_001.wav

## Parsed fields

speaker_id = M001  
language = EN  
source = HUMAN  
stimulus_id = EN_ADJ_001  

## Required rule

The final stimulus ID must match the stimulus_id field in the master stimulus spreadsheet.
