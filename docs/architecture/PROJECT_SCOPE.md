# Savlaq Project Scope

Savlaq is a Praat-centered automated speech-processing pipeline for acoustic phonetic research.

## Core purpose

Savlaq automates the transition from raw speech recordings to analysis-ready acoustic measurements.

## Current project use case

The first use case is a Journal of Phonetics-oriented study comparing human and AI/TTS speech in Turkish and General American English lexical prominence.

## Engine

Praat is the primary acoustic-processing engine.

Savlaq scripts will automate Praat rather than replace it.

## Planned workflow

Raw audio recordings  
→ file validation  
→ TextGrid creation  
→ segmentation support  
→ acoustic measurement extraction  
→ CSV export  
→ R-based quality control and statistical preparation

## First development principle

Do not automate interpretation before the basic file-processing pipeline is stable.
